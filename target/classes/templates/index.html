Bem vindo ao Negociador de Ações

Para utilizar a aplicação siga os passos abaixo:

1- Cadastre as contas que irão realizar os investimentos.

2- Cadastre as empresas que deseja monitorar a variação de valores das ações.

3- Cadastre regras de investimento que serão seguidas para os investimentos da conta.

4- Gere uma simulação para conferir os resultados.
